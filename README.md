# LOCPEW2 - US Media Headlines (July-September 2011)

This repository contains US mainstream media headlines from July 1, 2011 to September 30, 2011, collected from the Pew Research Center's Project for Excellence in Journalism (PEJ) News Coverage Index and the Library of Congress.

## Data Sources
- Pew Research Center's Project for Excellence in Journalism (PEJ) News Coverage Index
- Library of Congress

## Data Structure
- `data/` - Contains the collected media headlines data
  - `pew/` - Data from Pew Research Center
  - `loc/` - Data from Library of Congress
- `scripts/` - Python scripts for data collection and processing
